# Templates

Reusable template library for planning and execution discipline.

All management-facing templates should follow timestamp and estimation rules
from `../management/TIMELINE.md`.

## Available Templates

- `task-template.md`
- `plan-template.md`
- `review-template.md`
- `strict-review-template.md`
- `operational-doc-template.md`
- `how-this-works-template.md`
- `adr-template.md`
- `change-dod-template.md`
