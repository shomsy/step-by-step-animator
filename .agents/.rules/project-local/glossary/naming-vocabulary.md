# Naming Vocabulary

## Preferred Verbs

- `discover`, `read`, `validate`, `normalize`, `project`, `compile`, `build`, `play`, `show`, `present`, `create`, `find`, `write`

## Avoid As Default

- `handle`, `process`, `manage`, `doWork`, `helper`, `util`, `service`, `manager`

## Rule

Folder says flow.
File says responsibility.
Function says exact action.
