# Agents Tree

This directory holds the repo management contract. It is separate from the product and system code.

## Responsibility Map

- `architecture/` - repo shape, migration posture, and naming baseline
- `authoring/` - lesson authoring rules and source contracts
- `backlog/` - raw ideas and user stories that are not yet committed work
- `bugs/` - active defects, regressions, and hotfix triage
- `planning/` - selected workstreams, breakdowns, dependencies, and estimates
- `evidence/` - actionable TODOs and closed evidence
- `governance/` - quality rules, review gates, and project policy
- `review/` - review findings and review archive notes

## Reading Order

1. `architecture/`
2. `authoring/`
3. `governance/`
4. `backlog/`
5. `planning/`
6. `bugs/`
7. `evidence/`
8. `review/`

## Notes

- `planning/ARCHIVE.md` and `review/bugs.md` are historical records.
- `evidence/TODO.md` remains the active implementation queue.
- `evidence/CHANGELOG.md` remains the closed-state ledger.
