7. Observed Attributes and Sync - Dodajemo reaktivnost na promene atributa iz HTML-a.
8. Property API Through Host Controls - Povezujemo atribute sa JavaScript properties API-jem.
9. Host Events With Real Interaction Feedback - Host reaguje i na preset promene i na klik kroz custom event.
